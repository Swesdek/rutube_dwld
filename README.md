## Rutube downloader

# RU

## Использование

Если у вас установлен go, то вы можете использовать утилиту так:

````bash
go run ./cmd/main.go --threads 100 RUTUBE_LINK
````

Или же так, с помощью скомпилированного файла:

````bash
rutube_dwld.exe --threads 100 RUTUBE_LINK
````
